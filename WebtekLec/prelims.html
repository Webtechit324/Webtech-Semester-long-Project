Internet

- a global system of interconnected computer networks that use the standard Internet Protocol Suite (TCP/IP) to serce billions of users worldwide.

- It is a networks that consists of millions of private, public, academic, business and government networks, of local to global scope, that are linked by a broad array of electronic, wireless and optical networking technologies.

Vinton Cerf

- Father of the Internet

World Wide Web

- A global information medium which users can read and write via computers connected to the Internet. The term is often mistakenly used as a synonym for the Internet itself, but the Web is a service that operates over the Internet, as electronic mail (e-mail) does.

Tim Berners-Lee

- Father of the Web

- In September 1994, Tim Berners-Lee founded the World Wide Web Consortium (W3C) at the Massachusetts Institute of Technology with support from the Defense Advanced Research Projects Agency (DARPA) and the European Commision. It comprised various comapanies that were willing to create standards and recommendatiopns to improve the quality of the Web.

HTTP Request Methods

  • GET

    • Request data from a specified source.

  • HEAD

    • Same as get but returns only HTTP headers and no document body.

  • POST

    • Submit data to be processed to a specified resources.

  • OPTIONS

    • OPTIONS method represents a request for information about the communication options available on the request/response chain identified by the Request-URI.

  • TRACE

    • TRACE method is used to invoke a remote, application-layer loop- back of the request message..

  • PUT

    • Uploads a representation of the specified URI.

  • DELETE

    • DELETE method requests that the origin server delete the resource identified by the Request-URI.

  • CONNECT

    • Converts the request connection to a transparent TCP/IP tunnel
