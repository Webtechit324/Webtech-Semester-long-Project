CSS Selector

Selector

- elements in the document tree are matched

- Example: body ail# x>p-y[title] + span: +-before

- condition in a CSS rule: match elements called the subject of the selector

Selector Syntax

- one or more sequences of simple selector

- Example: body(simple) p.xyz (selector sequence) + h1[title]::before(pseudo element)

Sequence of Simple Selector

- chain of simple selector not separated by ;

- Example: h1,h2,h3 - group of selector

Simple Selector

- type selector - name of elements in html

- universal selector - represented by *{} (target everything)

- attribute selector - Example: p[attr]class='x y' and Attribute Selector

- class selector - Example: class = 'x y' -> p.x

- id selector - Example: id = 'id' -> #id

- pseudo class

- dynamic pseudo-class - Example: Link Pseudo-Class , User Action Pseudo-Class , Target Pseudo-Class , Language Pseudo-Class , UI Element States Pseudo-Class , Structural Pseudo-Class , Negation Pseudo-Class and Combinators
